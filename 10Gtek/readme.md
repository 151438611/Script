#10Gtek工作岗位使用<br>
<br>
使用方法：bash check-tools.sh <br>
------------------ 工作模式 ------------------
1-自动检查编码
2-手动输入订单号检查编码
3-创建兼容测试模板文件
4-整理排板邮件中的产品类型、SN
5-自动放码：只需放入Port1或Port2，将自动复制到其他Port; 若有Page02放一个并重命名为起始SN即可
6-创建ZQP-P02全FF的bin文件(适用于SN后4位为非数字编码工具无法生成的场景)---待测试
7-针对生产写码QSFP/4SFP、ZQP/4ZSP二端SN不一致无法写码，仅修改QSFP端SN命名，和SFP端SN保持一致

请选择工作模式,***直接回车***退出程序：
